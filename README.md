# 🍳 Cooksip Api

A RESTful API for managing recipes, ingredients, and users.
This project allows users to create, update, delete, and explore recipes with ingredients, steps, and tags.

Built with **Node.js**, **Express**, **TypeScript**, and **Prisma ORM**.

---

# 🛠️ Tech Stack

* **Node.js**
* **Express**
* **TypeScript**
* **Prisma ORM**
* **PostgreSQL**

---

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/figuedro/cooksip-backend.git
```

Enter the project directory:

```bash
cd recipe-api
```

Install dependencies:

```bash
npm install
```
